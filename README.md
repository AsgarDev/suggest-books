
# Anime Collection
A simple app that implements RabbitMQ to handle book suggestion message.

## Run Locally

Clone the project

```bash
  git@github.com:AsgarDev/suggest-books.git
```

Run the docker-compose

```bash
  docker-compose up -d
```
